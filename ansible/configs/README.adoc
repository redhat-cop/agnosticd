= Configs

This directory contains playbooks to provision, deploy, configure softwares. Each config should containsthose files:

* `pre_infra.yml`
* `post_infra.yml`
* `pre_software.yml`
* `software.yml`
* `post_software.yml`
* `env_vars.yml`
* `destroy_env.yml`

For more details see link:../main.yml[main.yml] and link:just-a-bunch-of-nodes[just-a-bunch-of-nodes].
