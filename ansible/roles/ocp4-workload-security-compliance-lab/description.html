OCP Starter Workshop

Starter Workshop is a workshop for a high number of users (<200) that teaches how to start developing on OpenShift. ​This workshop walks the audience through​ ​the process​ and concepts of developing an application on OpenShift by creating and deploying a microservices type of application.

Products and Projects:

  • OpenShift Container Platform
  • OpenShift Application Runtimes
    (Spring Boot)
  • JBoss EAP

​This demo deploys all the required infrastructure to deliver the workshop. It will create an infrastructure project (ocp-workshop) where you can

  • Nexus (for dependency caching)
  • Gogs (for Git hosting)
  • Firepad (for user assignment)
  • Workshopper (with all the workshop guides)

Every user will have the required source code in the Gogs platform deployed in the system

Also, every user will have a project already created for them, and they will have quota defined to prevent abuse

Source code for the guides: https://github.com/openshift-labs/starter-guides

Source code for the labs: https://github.com/openshift-roadshow

Provisioning Time: ~5 min
