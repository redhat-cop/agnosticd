Kasten K10 by Veeam Demo & Workshop

Kasten K10 is a Cloud Native data management platform for Day 2 operations

In this Red Hat OpenShift Container Platform (OCP) workshop, we'll explore how Kasten K10 provides enterprise DevOps teams with an easy to use, scalable, and secure system for backup/restore, disaster recovery and application mobility for Kubernetes applications.

With an application-centric approach and deep integrations with relational and NoSQL databases, storage systems, and Kubernetes distributions, Kasten K10 delivers flexibility and freedom of choice across cloud & on-prem infrastructures, virtual machines and container application stacks.

And because Kasten K10 provides data mobility and tranform capabilities, it can be leveraged by Red Hat OpenShift customers to move their cloud native workloads and data deployed on other Kubernetes platforms, whether on-premises, at the edge, or in the cloud to RedHat OpenShift.

This short workshop provides a fully deployed and running Katen K10 deployment on OCP and will cover the following topics:

  • Deploy an application with persistent data
  • Configure Kasten K10
  • Backup the application
  • Simulate a data loss event
  • Restore the application
