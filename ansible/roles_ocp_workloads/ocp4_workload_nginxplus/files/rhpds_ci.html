NGINX Demo & Workshop

Handling Cluster Traffic in OpenShift with the NGINX INgress Operator

In this Red Hat OpenSHift Container Platform (OCP) workshop, you will use the NGINX Ingress Operator to handle networking for apps running in OCP. This lab is 30 minutes, hands-on, and comes with the NGINX Ingress Operator Installed and licensed.
