OCP4 Starburst Enterprise Demo

Starburst Enterprise is a cloud-native, distributed SQL query engine that enables fast and scalable access to data across disparate sources.

Starburst Enterprise is the leader in providing a fast, scalable, and cost-effective solution for data access and analytics across multiple platforms.. Together, Starburst and Red Hat OpenShift help organizations build a comprehensive solution for deploying, and managing data analytics applications..

This workshop provides a Starburst Enterprise deployment on OCP and will cover the following topics:

  • Accessing the Starburst Web UI
    • Overview of Data Sources
  • Adding Additional Data Sources
  • Adding Data
