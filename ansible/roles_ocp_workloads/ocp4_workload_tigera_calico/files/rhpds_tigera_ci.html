Implementing Zero-Trust Security for Containers with Tigera Calico

In this Red Hat OpenShift Container Platform (OCP) workshop, you will work with Tigera Calico and OpenShift to implement Zero-Trust security for workloads running on OCP. This 90-minute hands-on lab comes with your own Calico environment.
